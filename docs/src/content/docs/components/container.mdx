## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Container content |
| `className` | `string` | — | Additional CSS class |

## Usage

```tsx
import { UIContainer } from "tango-api";

<UIContainer className="my-custom-section">
  {/* content */}
</UIContainer>
```
