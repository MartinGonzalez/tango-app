## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | required | Main message |
| `description` | `string` | — | Supporting text |
| `action` | `ReactNode` | — | Action element (button, link, etc.) |

## Usage

```tsx

<UIEmptyState
  title="Nothing here"
  description="Create something to get started."

/>
```
