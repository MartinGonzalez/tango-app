## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `options` | `Array<{ value: string; label: string }>` | required | Available segments |
| `value` | `string` | — | Currently active segment |
| `onChange` | `(value: string) => void` | — | Called when selection changes |

## Usage

```tsx
import { UISegmentedControl } from "tango-api";

<UISegmentedControl
  options={[
    { value: "list", label: "List" },
    { value: "grid", label: "Grid" },
  ]}
  value={viewMode}
  onChange={setViewMode}
/>
```
