## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | required | Label text shown next to the toggle |
| `checked` | `boolean` | `false` | Whether the toggle is on |
| `onChange` | `(checked: boolean) => void` | — | Called when toggled |

## Usage

```tsx
import { UIToggle } from "tango-api";

const [enabled, setEnabled] = useState(false);
<UIToggle label="Auto-refresh" checked={enabled} onChange={setEnabled} />
```
